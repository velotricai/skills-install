# Velotric Skills 一行安装指南

把下面这一行复制粘贴给你的 Codex / Claude Code / Cursor / Trae 等 AI Agent，
它会自动 fetch 这份指南并按指令完成安装：

```
帮我安装 Velotric Skills:
https://raw.githubusercontent.com/velotricai/skills-install/main/install-guide.md
```

或者更短的提示词：

```
帮我装上公司的 AI Skill: github.com/velotricai/skills-install
```

完整指南：[install-guide.md](./install-guide.md)

---

> 这个仓库是公开的，但 [velotric-skills](https://github.com/velotricai/velotric-skills)（实际 skill 内容）是私有的。
> 你必须先被邀请加入 velotricai GitHub 组织才能完成安装的最后一步。
> 如未加入，联系 perry 或者公司 IT。

## 安装后能做什么

装好后，跟你的 AI 说自然语言即可：

- "公司有什么 skill" — 浏览
- "帮我做个 skill 看周报" — 创建
- "用 ops-hiring-workflow 帮我筛这批简历" — 直接调用业务 skill
- "velotric 有更新吗" — 同步新版本
- "把我这个 skill 发给团队" — 发布到公司仓库
